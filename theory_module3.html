Моделирование и информатизация принятия решений

ОПРЕДЕЛЕНИЯ

  • Моделирование – процесс создания упрощённого представления (модели) системы или ситуации для анализа, прогнозирования и принятия решений.
  • Информатизация принятия решений – использование информационных технологий, данных и алгоритмов для поддержки и автоматизации процессов выбора оптимальных решений.

МОДЕЛЬ ПРИНЯТИЯ РЕШЕНИЯ

  1. Постановка проблемы, включая построение качественной модели процесса либо системы
  2. Конструирование концептуальной модели
  3. Выбор критерия эффективности
  4. Построение математической модели
  5. Выбор алгоритма оптимизации
  6. Численная реализация алгоритма
  7. Сбор данных и проверка модели
  8. Анализ полученных результатов и конструирование окончательного решения

ЭЛЕМЕНТЫ ЗАДАЧИ ПРИНЯТИЯ РЕШЕНИЯ

  • Лицо, принимающее решение (ЛПР) – человек или группа людей, ответственные за выбор
  • Проблемная ситуация – состояние системы, требующее вмешательства
  • Цель – желаемый результат
  • Альтернативы – возможные варианты действий
  • Последствия выбора альтернатив – прогнозируемые результаты
  • Признаки – характеристики для оценки альтернатив
  • Критерии – показатели предпочтительности
  • Предпочтения – субъективные приоритеты ЛПР
  • Принципы согласования оценок – правила выбора
  • Решение – выбранная альтернатива

КЛАССИФИКАЦИЯ ЗАДАЧ ПРИНЯТИЯ РЕШЕНИЙ

  1. По степени определённости информации: Общая задача, Задача выбора, Задача оптимизации
  2. По числу ЛПР: Индивидуальные, Групповые
  3. По количеству целей: Одноцелевые, Многоцелевые
  4. По содержанию: Технические, экономические, политические
  5. По времени: Краткосрочные, среднесрочные, долговременные
  6. По новизне: Стандартные, С аналогами, Уникальные
  7. По типу исхода: Детерминированные, Стохастические, Неопределённые
  8. По методу описания: Декларативные, процедурные, комбинированные
  9. По типу критериев: Точечные, интервальные, нечёткие
  10. По области применения: Управление, прогнозирование, диагностика

ПОСТАНОВКА ЗАДАЧИ ПРИНЯТИЯ РЕШЕНИЯ

Принятие решений – это процесс выбора наилучшего варианта действий из множества возможных альтернатив на основе заданных критериев и ограничений.
